# Linux Command Line #

## differences ##

1. RedHat : RedHat、Centos、Fedora等
2. Debian ：Debian、Ubuntu等

### ReHat ###
1. 常见安装包格式 rpm包 安装rpm 命令为 "rpm -参数"
2. 包管理器yum!!(yum install ......)
3. 支持tar包

### Debian ###
1. 常见安装包格式 deb包 安装deb 命令为 "dpkg -参数"
2. 包管理器 apt-get
3. 支持tar包
